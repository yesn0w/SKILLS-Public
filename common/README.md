# Common Assets

This directory is reserved for assets that are truly agent-agnostic.

There are no common skills in this repository yet. Helper scripts remain inside
the Codex and Claude skill packages that reference them so each installed skill
stays self-contained and its relative paths keep working.

Move an asset here only when the skill docs and installers are updated to use it
as shared agent-agnostic content.
